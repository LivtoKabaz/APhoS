About var4Astro

Tool for cataloguing astrophotometric data with the purpose of archiving variable star observations.

Application

Last updated 12.04.2022

The application parses and stores the csv data from the SIPS software. Its aim is to utilize all the photometrical data so each star is persisted in the database. The tool is work in progress. Current version provides no cross-identification between the catalogues. The advised catalogue to use is UCAC4.
